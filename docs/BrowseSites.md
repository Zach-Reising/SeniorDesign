# Browsing Sites

Sites are areas that users have identified need cleaning. Org owners or individuals can come here to view the locations and user/ai scores for how badly the area needs a touch-up.

### View a site
To view a site users can simple scroll to the site they wish and click view. Simple as that. When a user has cleaned a site they can report it as cleaned. Same goes for orgs.
