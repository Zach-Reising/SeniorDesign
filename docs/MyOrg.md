# My Org

The my org page allows a user to (if they've already create an org) manage their org

### Managing an Org

Org management has two parts
1) The org itself - to change org details simply click

    User supplied details of the org such as the name and active status can be clicked on to activate, typed over, and submitted to change. Currently allowed changes include

    - Org Name
    - Org Status

2) Org Users
    
    Org users are people with a role in your org. In this page a user can be right clicked on to open an actions menu and actioned by clicking one of the resulting options. Those options include.

    - A timeout from being able to view the org
    - A ban from the org
    - Role Promotion
    - Role Demotion
    - And More...
