# Org Search
Org search has two features
1) Create orgs link
2) Orgs browser

### Create Orgs Link
The create orgs link allows a signed in user to create their own org

When clicked the button will prompt the user for org details (name, area the org is based, etc.). Once filled out and submitted the org will be created for the user to use and grow.

### Browse Orgs
The orgs browser allows a user to join specific organizations and have push notifications about the orgs up and coming events sent directly to them. Simply scroll through the browser, view an org then press join. Thats all it takes.
